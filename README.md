# chainsaw_man_gesture_recognition
Using Python, OpenCV, and a custom MediaPipe hand recognition model to recognize contract and hybrid gestures from Chainsaw Man.

Installation guide
1. Clone repo
2. Build a virtual environment
# Create a virtual environment
python3 -m venv venv

# Activate it
source venv/bin/activate

# Install MediaPipe and other dependencies
pip install mediapipe opencv-python mediapipe-model-maker numpy tensorflow

# Navigate to your project
cd chainsaw_hand_gesture_recognition

# Create virtual environment
python3 -m venv venv

# Activate
source venv/bin/activate

# Install from requirements.txt
pip install -r requirements.txt
